Homestar Project Binary File Format
v1.1 / 02.07.2017

Header
------

4 bytes		| Format ID (HSPF) 0x48535046
4 bytes		| Header version 0x10002
n bytes		| Texture atlas name (null-terminated)

16x 4 bytes	| Customs colors (16x 24-bit RGB)

4 bytes		| Num of objects
n bytes		| Object structures

4 bytes		| Num of tiles
n bytes		| Tile structures

4 bytes		| Tilemap width
4 bytes		| Tilemap height
n bytes		| Tilemap data [4 bytes * width * height]

4 bytes		| Num of edges
n bytes		| Edge structures


Object structure
----------------

n bytes		| Object name (null-terminated)
4 bytes		| Object type
4 bytes		| Object Z
4 bytes		| Object color
4 bytes		| Num points
n bytes		| Points [4 floats: X, Y, U, V]
8 dwords	| Params

Tile structure
--------------

n bytes		| Tile name (null-terminated)
4 bytes		| Tile type
		|    0 = Full      (6 top UVs)
		     1 = Left      (4 top UVs)
                     2 = Right     (4 top UVs)
		     3 = Top       (3 top UVs)
		     4 = Bottom    (3 top UVs)
		     5 = Middle    (4 top UVs)
n bytes		| Top UVs 4-6x [2 floats: U, V]
n bytes		| Side UVs 4x [2 floats: U, V]
4 bytes		| Tile color

Edge structure
--------------

4 bytes		| Num of points
n bytes		| Points [int: Point number]
